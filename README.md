# To-do list 範例
### 使用技術
- React
- Apollo (Graphql)
- Ant-Design (UI Component)
- Styled-components (style)

### 架構
- todo-list-client (前端)
- todo-list-server (後端)
- lru-cache (資料庫)

### 啟動前端
```shell
cd todo-list-client
yarn install
yarn start
cd ..
```

### 啟動後端
```shell
cd todo-list-server
yarn install
yarn start
```
